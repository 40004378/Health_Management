# Health_Management

#Frontend Installation
Step1: Install node modules for both applications.
1. Healthmanagement
     "cd Healthmanagement"
     "npm install"
2. Tools_Catalog
     "cd Tools_Catalog"
     "npm install"
step2: To Run 2 applications.
1. Healthmanagement
    "npm start"
2. Tools_Catalog
    "npm start"
step3:To run JSON server

npx json-server --watch src/Components/Json/data.json --port 8000
